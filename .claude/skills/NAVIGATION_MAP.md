## ПРИНЦИПЫ НАВИГАЦИИ

### Принцип 1: НАЗАД — к предыдущему модулю, показать результат
Кнопка "Назад" показывает ПОСЛЕДНИЙ РЕЗУЛЬТАТ предыдущего модуля (не перегенерирует).

### Принцип 2: ВПЕРЁД — к следующему модулю, показать результат или запустить
Если следующий модуль уже генерировался — показать результат.
Если ещё нет — запустить генерацию.

### Принцип 3: Возврат НЕ стирает данные
Если я вернулся к стратегии и ОДОБРИЛ её (без изменений) — нейминг НЕ перегенерируется. Бот показывает существующий нейминг.

### Принцип 4: Перегенерация ТОЛЬКО при явном изменении
Перегенерация следующего модуля происходит только если пользователь ИЗМЕНИЛ (доработал) текущий модуль. Если просто посмотрел и одобрил — идём дальше без перегенерации.

### Принцип 5: Статус доступен всегда
На каждом экране — кнопка "📊 Статус" в последнем ряду inline keyboard.

## ВАЖНОЕ: ЛОГИКА "ПОКАЗАТЬ vs ПЕРЕГЕНЕРИРОВАТЬ"

### Функция showOrRun(module)
```
При переходе к модулю N:
  1. Проверить: есть ли артефакт модуля N?
     → Нет → запустить генерацию (run)
     → Да → показать последний результат (show) с полными кнопками
  
  2. При показе — кнопка "▶️ Вперёд" появляется только если
     следующий модуль (N+1) уже имеет артефакт
     
  3. Кнопка "↩️ Назад" появляется всегда (кроме модуля 1)
```

### Функция handleSmartApprove(module)
```
При одобрении модуля N:
  1. Пометить артефакт как approved
  2. Проверить: есть ли артефакт модуля N+1?
     → Да → Был ли модуль N ИЗМЕНЁН в этой сессии? (не просто просмотрен)
       → Да → "Данные изменились. Обновить [модуль N+1]?" [Да] [Нет]
       → Нет → Показать существующий результат модуля N+1
     → Нет → Запустить генерацию модуля N+1
```

### 1. Новый общий модуль: src/modules/navigation.ts

```typescript
// Определяет: показать существующий результат или запустить генерацию
async function showOrRunModule(ctx, moduleNum) {
  const artifact = getLatestArtifact для этого модуля
  if (artifact && artifact.status !== 'superseded') {
    // Показать существующий
    await showModuleResult(ctx, moduleNum, artifact)
  } else {
    // Запустить генерацию
    await runModule(ctx, moduleNum)
  }
}

// Показывает результат модуля с правильными кнопками
async function showModuleResult(ctx, moduleNum, artifact) {
  // Определить: есть ли следующий модуль с артефактом?
  const hasNext = moduleNum < 6 && await getLatestArtifact(nextModule)
  // Построить клавиатуру с учётом контекста (назад/вперёд)
  const kb = buildModuleKeyboard(moduleNum, artifact.id, hasNext)
  await sendLongMessage(ctx, artifact.data.text)
  await ctx.reply("Что делаем?", { reply_markup: kb })
}
```
