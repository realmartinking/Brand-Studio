## Linear

Основан в 2019 году. Keyboard-first, экстремально быстрый, принципиально мнение имеющий project management tool. Используется ведущими стартапами и engineering teams. Оценка $400M+. Философия: инструменты должны быть быстрыми, авторитетными и красивыми — противоположность enterprise software. Каждый кейстрок — меньше 100мс. Это не просто фича, это бренд.

## Why this position

Speed 0.1: Самый быстрый инструмент в датасете. Скорость — главный brand claim. «Built for speed» буквально.

Control-release 0.8: Сильно мнение имеющий (opinionated) — Linear диктует как организовывать работу. Нет бесконечных настроек. Одна правильная структура.

Nature-technology 0.9: Чисто технологический — абстракция данных и workflows. Никакой природной метафоры.

Serious-playful 0.8: Полностью серьёзный. Это профессиональный инструмент. Нет игривого onboarding.

All_few 0.6: Не для всех — дизайн подходит engineering-first командам. Масс-маркет (Jira) намеренно отвергнут.

## Attractor difference

Silicon-optimism: tech=0.9, serious=0.7, quiet=0.6, few=0.5. Linear совпадает по tech-ориентации и серьёзности. Ключевое отличие: speed=0.1 — Linear это extreme tech performance как главная ценность. ## Visual code

- **Typography:** Geist Sans (Linear's own font) — экстремально чистый geometric sans. Монострочный код как дополнение. Никаких serif, никакого декора.
- **Palette:** Тёмный фон (#111) как доминанта — dark mode как главный режим. Акценты — фиолетовый (#5E6AD2 Linear purple), белый текст.
- **Geometry:** Идеально прямые линии. Пиксельная точность. Никаких скруглений без причины.
- **Density:** Высокая — интерфейс насыщен issues, cycles, projects. Но иерархически выстроен.
- **Textures:** Экранный пиксель. Монохромный gradient в темноте. Код как визуальный элемент.
