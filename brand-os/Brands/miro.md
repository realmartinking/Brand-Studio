## Miro

Основан в 2011 году в Перми, Россия (как RealtimeBoard). Переименован в Miro в 2019. Онлайн-доска для удалённых команд — бесконечное визуальное пространство для совместной работы. 60M+ пользователей. Популяризировал remote collaboration во время пандемии COVID-19. Позиционирование — визуальное рабочее пространство для распределённых команд: стикеры в бесконечном масштабе.

## Why this position

Participation 0.9: Максимально co-created — продукт работает только когда несколько людей одновременно. Real-time collaboration как core value.

Self-community 0.7: Сильный community вектор — команды, не индивидуалы. Продукт создан для совместного мышления.

All_few 0.2: Массовый продукт — freemium, доступен любой команде. Демократизация visual collaboration.

Control-release 0.3: Release — бесконечная свобода доски. Нет жёсткой структуры (в отличие от Linear или Notion).

Repeatability 0.1: Ежедневный инструмент — открыт на каждом совещании. Максимальная рутинность.

## Attractor difference

Creator-economy: participation=0.9, technology=0.7. Miro совпадает по participation. Ключевое отличие: self_community=0.7 — Miro значительно более community-ориентирован чем типичный creator-economy (который более individual). Это collaboration tool, а не creation tool. ## Visual code

- **Typography:** Округлый дружелюбный гротеск — собственный шрифт или близкий к Circular. Тёплый, не холодный tech.
- **Palette:** Жёлтый (главный акцент, приближается к Mailchimp yellow), синий, белый. Оптимистичная рабочая палитра.
- **Geometry:** Скругления. Дружелюбные формы. Стикеры — квадрат со скруглёнными углами.
- **Density:** Низкая в маркетинге. Очень высокая в самом продукте — доска наполняется.
- **Textures:** Виртуальные стикеры, маркерные линии, цифровые рамки. Физические офисные метафоры в цифре.
