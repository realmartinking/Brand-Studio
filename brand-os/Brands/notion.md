## Notion

Основан в 2016 году. All-in-one workspace — заметки, базы данных, вики, задачи, документы. Оценка $10B на пике. Культовый среди indie-разработчиков, стартапов и knowledge workers. Знаменит бесконечно кастомизируемым интерфейсом — «второй мозг» и «командная вики» одновременно. Позиционирование — чистый холст который становится чем угодно.

## Why this position

Participation 0.7: Notion сам по себе — пустое пространство. Пользователь создаёт смысл. Продукт co-created by definition.

Nature-technology 0.8: Чисто цифровой продукт с органической метафорой (страница, блок, документ). Технология притворяется бумагой.

Repeatability 0.1: Ежедневный инструмент — открыт постоянно. Максимально рутинный.

Density 0.7: Интерфейс может быть экстремально насыщенным — сложные базы данных, вложенные страницы.

Self-community 0.5: Сильное community — Notion templates, Twitter/X power users, YouTube каналы посвящённые Notion-системам.

## Visual code

- **Typography:** Inter — один из самых нейтральных и читаемых интерфейсных шрифтов. Чистый, не претендующий. Бренд-шрифт — тот же Inter, узнаваемый через нейтральность.
- **Palette:** Белый + чёрный как базис. Бежевый/кремовый как «тёплый режим». Минимум акцентных цветов в UI.
- **Geometry:** Блочная система — всё из блоков. Бесконечная страница вниз. Иерархия через отступы.
- **Density:** Пользователь определяет. Базово — очень низкая (чистый лист).
- **Textures:** Экранные — имитация бумаги в тёплом режиме. Никаких физических референций.

## Verbal code

- **Tone:** Дружелюбный и вдохновляющий. Говорит о потенциале инструмента, не о конкретных функциях.
- **Phrase length:** Короткие для продукта, длинные для editorials о productivity.
- **Vocabulary:** Write, plan, organize, system, workspace, wiki, database, team, thought, build.
- **Examples:** «One workspace. Every team.» / «The connected workspace where better, faster work happens.» / «Your wiki, docs, and projects. Together.»
