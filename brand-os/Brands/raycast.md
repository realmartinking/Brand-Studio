## Raycast

Основан в 2020 году. macOS лаунчер и command palette — заменяет Spotlight с радикально расширенными возможностями. Сотни интеграций через extensions. Любимец разработчиков и power users. Позиционирование — твой Mac, но быстрее: командный интерфейс для того, как ты на самом деле думаешь.

## Visual code

- **Typography:** System font (SF Pro на macOS) — полная интеграция с OS. Моноширинный код для команд.
- **Palette:** Тёмный (#1A1A1A), свечение акцентов (#FF6363 красный, #6E56CF фиолетовый). Dark mode only в продукте.
- **Geometry:** Прямоугольный search bar — одна форма. Список результатов. Экстремально простая структура.
- **Density:** Низкая в UI (чистая строка) — высокая в функциональности (тысячи команд).
- **Textures:** Матовое стекло macOS blur, тёмный фон, текст как единственный элемент.
