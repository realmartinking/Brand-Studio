## Readymag

Основан в Москве. Браузерный инструмент для создания веб-публикаций, портфолио и editorial-дизайна. Используется фотографами, дизайнерами, культурными институциями. Инструмент выстроен вокруг идеи web как дизайн-медиума — не CMS, а визуальная среда. Глобальная аудитория, российское происхождение.

## Why this position

Nature-technology 0.8: Чисто цифровой инструмент, но с выраженной editorial эстетикой — ближе к печатному дизайну, чем к web dev.

Participation 0.5: Инструмент создания — пользователь строит публикации. Средняя co-creation (vs [[figma]] с 0.9 — Readymag более linear, менее collaborative).

All_few 0.5: Нишевый инструмент для профессиональных дизайнеров — не для всех, но не ultra-exclusive.

Rhythm 0.6: Ритм публикаций — выпусков, портфолио обновлений. Ритм редакции.

## Attractor difference

Creator-economy: participation=0.9, technology=0.7. Readymag совпадает по co-creation и tech. Ключевое отличие: более editorial и менее collaborative чем типичный creator-economy. Audience — cultural designers, не product teams. Ближайший дополнительный — [[intellectual-craft]]: та же аудитория и серьёзный подход к web как к медиуму.

## Visual code

- **Typography:** Тонкий editorial-ориентированный гротеск — Graphik, Suisse. Инструмент поддерживает любые шрифты пользователя — это само по себе позиция.
- **Palette:** Белый, чёрный, один фирменный цвет сезона. Neutral — чтобы не конкурировать с работами пользователей.
- **Geometry:** Свободная сетка — полная позиционная свобода для пользователя. Бренд сам по себе — минималистичный.
- **Density:** Средняя в коммуникации. Портфолио работ пользователей — насыщенные.
- **Textures:** Экранная типографика. Hover-анимации. Скролл как текстура.

## Verbal code

- **Tone:** Кураторский. Говорит как арт-директор — про визуальную культуру, не про функции.
- **Phrase length:** Средние editorial фразы.
- **Vocabulary:** Design, web, publish, visual, story, editorial, portfolio, layout.
- **Examples:** «Design for the web, not around it.» / «Every story deserves a canvas.» / «Publish with intention.»
