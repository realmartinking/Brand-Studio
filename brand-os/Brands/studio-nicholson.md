## Studio Nicholson

Основан Ником Уэйкманом в Лондоне. Портновская чёткость, применённая к расслабленному гардеробу. Японские ткани, лондонский крой. Гендерно нейтральный подход. Нет сезонной срочности — коллекции выходят по готовности. Позиционирование — продуманный портновский гардероб для ежедневной жизни, антимодная fashion-позиция.

## Visual code

- **Typography:** Нейтральный широкий гротеск — Neue Haas Grotesk, Helvetica. Много трекинга. Только строчные. Белое поле.
- **Palette:** Тёплый серый, сланцевый синий, экрю, умбра. Нейтральные японские тона — ни одного «корпоративного» цвета.
- **Geometry:** Прямоугольные силуэты с мягкостью. Конструкция видна в форме — не в декоре.
- **Density:** Средне-высокая в ткани — низкая в коммуникации. Lookbook = два образа, белый фон.
- **Textures:** Японская шерсть double-face, хлопок Oxford-weave, тёплый нубук.
