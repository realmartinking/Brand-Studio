## Things 3

Cultured Code, Штутгарт, Германия. Основан в 2007. Задачник для Apple платформ — iOS, iPadOS, macOS, watchOS. Победитель Apple Design Award 2017. Самый красивый task manager в экосистеме. One-time purchase — никакой подписки, принципиально. Позиционирование — задачи как спокойная, обдуманная практика: Marie Kondo продуктивности.

## Why this position

Loud-quiet 0.9: Радикально тихий. Никакой маркетинговой агрессии. Один app на Appstore, один сайт, редкие обновления. Продукт говорит сам.

Control-release 0.7: Things диктует GTD-inspired workflow. Но не принудительно — система мягкая.

Density 0.8: Информационная плотность в списке задач — максимальная функциональность без визуального шума.

Rhythm 0.7 / Repeatability 0.1: Ежедневный ритм — утренний просмотр задач как ритуал. Максимально рутинный.

Participation 0.1: Пассивное получение инструмента. Нет community features, нет collaboration.

## Visual code

- **Typography:** San Francisco (системный Apple шрифт) — полная интеграция с платформой. Нет кастомного шрифта — это принцип.
- **Palette:** Белый + очень светлый серый + синий акцент (#007AFF Apple blue) + тёплый кремовый для Today. Минималистичная Apple-native.
- **Geometry:** Прямоугольные ячейки задач. Круглые чекбоксы. Иерархическая отступность. Чистейшая сетка.
- **Density:** Высокая в списке задач — низкая в визуальном шуме. Information dense but visually calm.
- **Textures:** Матовое стекло iOS. Нативные элементы платформы. Никакой кастомной графики.

## Verbal code

- **Tone:** Практически молчаливый. Things 3 не говорит — делает.
- **Phrase length:** Технические release notes. Минимум маркетинга.
- **Vocabulary:** Task, project, area, today, deadline, do, focus, clear, next.
- **Examples:** «The to-do list for Mac, iPad, iPhone, and Apple Watch.» / «Collect. Organize. Act.» / «Clarity is the new productivity.»
