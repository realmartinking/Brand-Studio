## Windows

Microsoft's operating system platform, first launched 1985, commanding approximately 70% of global desktop OS market share. The Windows 11 logo redesign — executed by Pentagram — updated the four-pane window symbol: the four colored squares (the original "flag" logo) were replaced by a single flat four-pane window in a uniform gradient blue, arranged symmetrically without perspective distortion. The redesign moved Windows away from the skewed-perspective XP-era logo toward a cleaner, more architecturally precise form. Windows is mass-market infrastructure software — the default computing environment for most of the world's businesses and households.

## Why this position

All-few 0.1: Windows is the definition of mass distribution. Over a billion active devices. No brand in this system has a lower all_few score.

Self-community 0.3 → 0.4: Windows is used individually but in workplace and school contexts where the OS is shared infrastructure. The community is the organisation, not the brand.

Control-release 0.6: Microsoft controls the hardware certification, the store, the update cycle — but Windows as a platform deliberately allows wide diversity of software, hardware manufacturers, configurations. More open than Apple, more controlled than Linux.

Body-mind 0.7: The OS mediates everything physical — keyboard, mouse, touch, stylus on Surface. Interaction is inherently embodied. But the brand has moved toward mental productivity (Microsoft 365, Copilot AI) in recent cycles.

Roots-horizon 0.5: Windows carries 40 years of history — the blue screen of death, Clippy, Solitaire — but actively promotes itself as evolving (Windows 11, Copilot integration). Genuine tension between legacy and renewal.

Distance 0.3: Windows is approachable by design — the whole interface philosophy since Windows 95 has been accessibility for non-technical users. The brand speaks with relative warmth for an infrastructure product.

Repeatability 0.1: Daily constant — Windows is on for hours every day for most users.

## Attractor difference

friendly-helper is the nearest attractor for Windows in its consumer register — the brand aspires to be approachable, accessible, helpful, and warm. But the similarities are surface-level. Windows differs from the friendly-helper ideal in critical ways: it is not chosen by users (it is the default), it serves institutional buyers as much as individual users, and its quality is not universally praised. The institutional and enterprise weight pulls toward [[corporate-trust]], while the OS infrastructure positioning (system-level, invisible-when-working) echoes [[bauhaus-functional]]. Windows is also more neutral in temperature and less participative than a true friendly-helper brand — it doesn't invite you to engage, it just runs.

## Visual code

- **Typography:** Segoe UI (Windows system typeface) — a humanist sans-serif designed by Steve Matteson for Microsoft, optimised for screen legibility. Medium weight, neutral, slightly warm.
  *Source: training-data*
- **Palette:** Windows Blue — in Windows 11, a gradient from #0078D4 (Microsoft Blue) through lighter sky tones. The four-pane window logo uses a single blue-to-light-blue gradient. Secondary brand palette includes Microsoft product colors (Teams purple, Office red, etc.).
  *Source: training-data*
- **Geometry:** The Windows logo: four squares arranged 2×2, symmetrical, flat, no perspective skew (Windows 11 update). Clean grid geometry. The Fluent Design language uses rounded corners, acrylic blur effects, and soft shadows in the UI.
  *Source: training-data*
- **Density:** Marketing is typically low density — product hero shots, lifestyle photography, generous space. OS UI itself is medium density with taskbar, widgets, app grid.
  *Source: training-data*
- **Textures:** In OS UI: acrylic/mica materials (translucent blur effect introduced in Windows 11). Marketing: photography-forward, no texture.

## Verbal code

- **Tone:** Functional, accessible, occasionally aspirational. The brand does not try to be cool — it tries to be reliable and useful.
- **Phrase length:** Short to medium. Product-forward. Benefit-led.
- **Vocabulary:** Productivity language (do more, create, achieve, connect), accessibility claims (for everyone, designed for you), innovation signals (new, reimagined, built for), universal address (you, your).
- **Examples:** "Windows 11. Designed for hybrid work." / "More ways to be you." / "Hello again."
